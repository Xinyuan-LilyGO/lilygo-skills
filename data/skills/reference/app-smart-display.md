# Recipe Context: Smart Display

Use as source and recipe context for dashboard, kiosk, panel UI, and always-on
smart display prompts. This skill does not prove rendering or touch behavior.

- Pair with display, LVGL, storage, power, and the selected framework.
- Prefer CLI-buildable examples and evidence scripts over GUI-only workflows.
- Rendered UI success requires simulator, page-data, screenshot, or board evidence.
