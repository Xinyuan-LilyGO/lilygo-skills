# Tool: PlatformIO CLI

Use for PlatformIO Core, `platformio.ini`, `pio run`, upload, monitor, and environment prompts.

- Primary authority: `https://docs.platformio.org/en/latest/core/index.html`.
- Pair LilyGO PlatformIO work with `fw-platformio` and the underlying Arduino or ESP-IDF framework.
- Build/upload success requires command output.
