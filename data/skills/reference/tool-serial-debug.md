# Tool: Serial Debug

Use for serial port discovery, boot logs, read/write, RTS/DTR, monitor, and serial evidence prompts.

- Skill reference: `serial-debug`.
- Project reference: `https://github.com/Adancurusul/serial-mcp-server`.
- Record port, baud, command, exit code, and a bounded log excerpt for evidence.
- Do not change RTS/DTR unless the command explicitly asks for it.
